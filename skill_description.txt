This skill reads a list of up to 25 games in a specified tag (such as adventure, horror, co-op, etc.)  according to the following filters: popular new releases, top sellers, or specials, as according to Steam store.

Supports up to 330 most popular game tags as found here (as of 12/26/2016): http://store.steampowered.com/tag/browse/#global_492

Invoke the skill in the following format:
Alexa, ask hot dog for [filter] in [tag].

Games can be searched without tags if no tag is provided or a tag of "overall" is explicitly provided.

Alexa will read the first 25 games following the specified criteria in the format of [title], [discount %], [final price]
